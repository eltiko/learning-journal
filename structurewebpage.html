INTROCUCTION TO CSS

CSS is a language to give style to a html page.
It allows us to choose the appearance of our page from selectors and declaration.
With css it is possible to modify the position of the texe, its size, its shape its color ...

For example the following code makes it possible to have a text with the red color.

p. { font-color: red; }

How to style a web page with colors

First of all it is necessary to select the part of the document or the color is applied.
For example 'p' allows you to select a paragraph.
Then we open curving braces and type front-color followed by a color name.
